Cellular Automata

See this page for more information about what this program is all about.

Click "Draw New" to begin with a random rule and random colors. "Refresh Current" will keep the same rule and the same colors from the previous drawing (but any change in the number of colors is ignored), but adjust the other options if they have been changed. "Reset" returns the form elements to their defaults, but doesn't redraw the image. The text box below gives the rule that was used to draw the picture (according to my previous convention for describing rules), and the time it took to draw it.

Warning: the speed of this program is rather slow. Therefore I don't recommend setting Magnification = 1, or making the image too large, unless you don't mind waiting.

You'll need a modern browser to view the app. It has been tested in Firefox (Windows/Linux) and Chrome (Windows).
