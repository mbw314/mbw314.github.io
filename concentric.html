Home / Programming / Concentric Polygons

The purpose of this program is to animate an "infinite zoom" into a sequence of concentric regular polygons. By default, all polygons share the same orientation, but if rotation is enabled, then each one is rotated by an adjustable offset angle relative to its neighbors to avoid overlaps. The polygons can be draw simply as a collection of line segments, or filled with random colors.
