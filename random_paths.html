Random Paths in a Graph

In this HTML canvas experiment, I wanted to create random paths in a graph: choose a vertex randomly, move along an edge to a random neighboring vertex, etc. until there are no possible unoccupied vertices. Then start again with a new random vertex, and repeat the process until all vertices of the graph are occupied.

The process is animated below, and there are a few types of graphs to choose from: rectangular, triangular, or hexagonal grids.

You'll need a modern browser to view it (e.g. Firefox, Chrome, Safari).
